## For PLWDs (Users)

### Registration Steps:

1. Click **"Register Now"** button on homepage
2. Fill in:
    - Full Name
    - Email Address
    - Password
    - Confirm Password
3. Click **"Register"**
4. You'll be redirected to your dashboard

### Complete Your Profile:

1. Click **"Edit Profile"** in the menu
2. Fill in all required fields:
    - Gender
    - Date of Birth
    - Phone Number
    - Address
    - State and LGA
    - Disability Type
    - Education Level
    - Skills (check all that apply)
3. Upload a profile photo (optional)
4. Click **"Save Profile"**

### Upload Documents:

1. Go to your dashboard
2. Scroll to **"Uploaded Documents"** section
3. Click **"Upload Document"**
4. Select document type (ID Card, Medical Report, etc.)
5. Choose file (PDF, JPG, PNG - Max 5MB)
6. Click **"Upload"**

### Check Verification Status:

-   Your profile status is shown on the dashboard
-   **Pending** = Waiting for admin approval
-   **Verified** = Your profile has been approved

### Manage PLWDs:

1. Click **"Manage PLWDs"** in sidebar
2. Use filters to search:
    - By name/email
    - By state
    - By disability type

### Approve/Reject Profiles:

1. Click **"View"** button on any PLWD
2. Review their profile and documents
3. Click **"Approve Profile"** or **"Reject Profile"**
4. An email will be sent to the user (if configured)

### Export Data:

1. Go to **"Manage PLWDs"**
2. Apply desired filters
3. Click **"Export Excel"** or **"Export PDF"**
4. File will download automatically

### View Audit Logs:

1. Click **"Audit Logs"** in sidebar
2. See all admin actions with timestamps

### Manage Metadata:

-   **Disability Types**: Add/edit disability categories
-   **Education Levels**: Manage education options
-   **Skills**: Add/edit available skills

## Troubleshooting

### Can't Login?

-   Check your email and password
-   Use "Forgot Password" link
-   Contact admin if registered as PLWD
-   Contact system administrator if admin

### Profile Not Verified?

-   Wait for admin review (usually 1-3 days)
-   Ensure all required fields are completed
-   Upload all required documents
-   Check email for feedback

### Upload Failed?

-   Check file size (max 5MB)
-   Verify file type (PDF, JPG, PNG only)
-   Try a different file
-   Contact admin if problem persists

### Can't Export Data?

-   Ensure you have admin role
-   Check your browser's download settings
-   Try a different browser
-   Contact technical support
